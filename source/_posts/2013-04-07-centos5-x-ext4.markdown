1、去/lib/modules/2.6.18-308.el5xen/kernel/fs/ext4/，看一下是ext4.ko还是ext4dev.ko，
然后modprobe ext4或者modprobe ext4dev

2、然后需要安装一个包
```
# yum install -y e4fsprogs
```
之后就可以使用mkfs.ext4了。
