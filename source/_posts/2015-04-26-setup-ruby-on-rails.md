安装ROR真费劲。。

1. 安装ruby

```
sudo yum install ruby
```

2. 安装ruby-devel

```
sudo yum install ruby=devel
```

3. 安装rails

```
sudo gem install rails
```

可能需要设置gem的资源服务器为淘宝的。

最好再安装上，mechanize，

```
sudo gem install mechanize
```
